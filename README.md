# Typing Speed Test

Measure your typing speed (WPM) and accuracy in the terminal with multiple difficulty levels.

## Features

- 4 difficulty levels: Easy, Medium, Hard, Code
- WPM (Words Per Minute) and CPM (Characters Per Minute) calculation
- Accuracy tracking with character-by-character diff (color-coded)
- Practice mode for continuous typing
- Persistent high scores
- Performance rating

## Usage

```bash
python main.py
```

No external dependencies required!

## Difficulty Levels

| Level | Content | Description |
|-------|---------|-------------|
| Easy | Short sentences | Common phrases and proverbs |
| Medium | Longer sentences | Quotes and longer passages |
| Hard | Technical paragraphs | DevOps and cloud computing topics |
| Code | Programming snippets | Python code snippets |
